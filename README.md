# Pilhas-atividade

Projeto acadêmico focado na implementação e manipulação da estrutura de dados Pilha (Stack) utilizando a linguagem Java. O objetivo deste repositório é demonstrar o conceito LIFO (Last-In, First-Out), abordando desde a criação da estrutura base até a aplicação de métodos para inserção, remoção e verificação de elementos, aplicando conceitos sólidos de Programação Orientada a Objetos.
